# eat-the-burger

## Usage
---
1.  Use schema.sql and seeds.sql to insert some burgers into your MySQL database.
2.  Then, use 
    'node server.js' 
    to run the server and have it listen on localhost port 8080.
3.  Navigate to localhost:8080 in your browser and use the Eat the Burger app!
